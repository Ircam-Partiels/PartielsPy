property format: str¶

Export format of the image (jpeg or png).

Set to a valid format string to change it. The default format is jpeg.

property groups: bool¶

Whether the groups option is set.

Set to a boolean to change it. This option is used to set the group flag in the command line. The default value is False. If set to True, the images of the groups are exported.

property height: int¶

Whether the image height is set.

Set to a positive integer to change it. The default height is 720 pixels.

property width: int¶

Whether the image width is set.

Set to a positive integer to change it. The default width is 1280 pixels.
