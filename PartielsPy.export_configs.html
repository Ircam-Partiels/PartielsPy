Export base configuration class

This is the base class used to configure the export of the document. Is not intended to be used directly but inherited by the ExportConfig classes: - PartielsPy.export_configs.image - PartielsPy.export_configs.lab - PartielsPy.export_configs.reaper - PartielsPy.export_configs.text_base
